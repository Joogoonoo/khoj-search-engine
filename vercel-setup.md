# Vercel CLI सेटअप

## CLI इंस्टॉल करें
```bash
npm i -g vercel
```

## Vercel प्रोजेक्ट के साथ लिंक करें
```bash
vercel login
vercel link
```

## लोकल डेवलपमेंट के लिए प्रोजेक्ट चलाएं
```bash
vercel dev
```

## प्रोडक्शन बिल्ड और डिप्लॉय करें
```bash
vercel
```

## प्रोडक्शन के लिए डिप्लॉय करें
```bash
vercel --prod
```
